# pi extensions

Custom extensions auto-loaded by pi via the `extensions` array in
[`../settings-baseline.json`](../settings-baseline.json). Each `.ts` file is the source of truth; the sibling `.md`
(where present) is a deep reference for the behaviour, rule shape, and environment variables.

Add new extensions to this directory, import their default export from `settings-baseline.json`, and run `/reload` in an
interactive pi session to pick up changes without restarting.
